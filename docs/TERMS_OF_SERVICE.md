# ToDaDot 서비스 약관

**최종 업데이트: 2025년 12월 12일**

## 1. 서비스 개요

ToDaDot은 아이젠하워 매트릭스를 활용한 투두리스트 및 캘린더 관리 데스크톱 애플리케이션입니다.

## 2. 서비스 이용

### 2.1 사용 조건
- ToDaDot은 무료로 제공되는 오픈소스 소프트웨어입니다.
- 사용자는 본 약관에 동의함으로써 서비스를 이용할 수 있습니다.

### 2.2 데이터 저장
- 모든 데이터는 사용자의 로컬 컴퓨터에 저장됩니다.
- ToDaDot은 사용자 데이터를 외부 서버에 저장하지 않습니다.
- Google Calendar 또는 Firebase 연동 기능을 사용하는 경우, 해당 서비스의 약관이 적용됩니다.

## 3. 책임의 제한

### 3.1 데이터 손실
- ToDaDot은 사용자 데이터의 백업을 자동으로 수행하지 않습니다.
- 사용자는 자신의 데이터에 대한 책임을 집니다.
- 데이터 손실에 대한 책임은 사용자에게 있습니다.

### 3.2 서비스 중단
- ToDaDot은 오픈소스 프로젝트이며, 서비스 중단 가능성이 있습니다.
- 개발자는 서비스의 지속적인 제공을 보장하지 않습니다.

## 4. Google API 사용

### 4.1 Google Calendar 연동
- Google Calendar 연동 기능을 사용하려면 Google OAuth 2.0 인증이 필요합니다.
- Google API 사용 시 Google의 서비스 약관이 적용됩니다.
- Google API 사용에 대한 책임은 사용자에게 있습니다.

### 4.2 Firebase 연동
- Firebase 연동 기능을 사용하려면 Firebase 설정이 필요합니다.
- Firebase 사용 시 Google의 서비스 약관이 적용됩니다.
- Firebase 사용에 대한 책임은 사용자에게 있습니다.

## 5. 오픈소스 라이선스

ToDaDot은 MIT 라이선스 하에 배포됩니다.

## 6. 약관 변경

본 약관은 사전 고지 없이 변경될 수 있습니다.

서비스 약관에 대한 문의사항이 있으시면 GitHub 리포지토리의 Issues를 통해 문의해주세요.
